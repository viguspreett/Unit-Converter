# Unit-Converter
A Python GUI Based unit converter built using Tkinter.
Convert values between common units quickly using an intuitive interface.

# Features
1. Graphical user interface (Tkinter)
2. Input-based value conversion
3. Supports multiple unit conversions
4. Instant result display
5. Lightweight and beginner-friendly

# Tech Stack
Python and Tkinter

# How to Run
1) Clone the repository
2) Navigate the project directory
3) Run the script

# Use Case
- Learning Python GUI development
- Practicing logic for unit conversions
- Small utility project for beginners
